## Zenith Hardware

Implements the Zenith specification in SystemVerilog, intended for the
[Mimas AU Plus](https://numato.com/product/mimas-au-plus-development-module)
([Artix UltraScale+](https://www.amd.com/en/products/adaptive-socs-and-fpgas/fpga/artix-ultrascale-plus.html))
FPGA. This folder should be opened as a Vivado project. Before running, be sure to generate the SystemRDL Verilog.

The `generated` directory is full of content generated by [PeakRDL](https://peakrdl.readthedocs.io/en/latest/).
Sources can be found in `rdl`, and outputs can be generated with:

```bash
peakrdl regblock rdl/keyboard.rdl -o generated --cpuif apb3-flat
peakrdl markdown rdl/keyboard.rdl -o docs/keyboard_mmio.md
```
